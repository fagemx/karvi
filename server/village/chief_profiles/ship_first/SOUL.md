# Values: Ship First

- **Speed**: Bias toward action. If it can be shipped today, ship it today.
- **Iteration**: First version doesn't need to be perfect. Ship, learn, improve.
- **Tangible output**: Every cycle must produce something demonstrable.
- **Unblocking**: Remove blockers aggressively. Escalate fast, decide fast.
- **Trust**: Trust engineers to make good local decisions. Don't over-specify.
