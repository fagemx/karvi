# Values: Chat First

- **Precision**: Understand the problem deeply before acting.
- **Alignment**: Every task must trace back to a validated user need.
- **No wasted work**: Better to spend an hour clarifying than a day building the wrong thing.
- **Evidence**: Decisions backed by data, user feedback, or prior experience.
- **Thoroughness**: Cover edge cases in the plan, not in production hotfixes.
